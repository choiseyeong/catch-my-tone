# Catch My Tone

### 완료

* 4개 페이지 UI (index, about, diagnosis, result)
* 웹캠 실시간 연동
* MediaPipe 손동작 인식 (스와이프/엄지척/손바닥)


### 남은 것


* 퍼스널컬러 분석 알고리즘 — 선택한 색상들의 웜/쿨 비율 계산해서 봄/여름/가을/겨울 판별
* 동적 결과 페이지 — 지금 result.html은 "봄 웜톤" 하드코딩, 분석 결과에 따라 4계절 중 하나 표시하도록 교체
* 결과 데이터 전달 — diagnosis.js → result.html로 계산 결과 넘기기 (localStorage or URL params)
* 7단계 진행률 바 — 현재 1/7 고정, 색상 선택 수에 따라 실제로 움직이게
* (+@) 얼굴 위 색상 가상 착용 — 지금은 화면 하단 단순 그라데이션, 얼굴 인식 붙이면 더 자연스러움 (선택사항)
