# Catch My Tone

### 프로젝트 소개

이 프로젝트는 OpenCV 손동작 인식 기반 퍼스널컬러 진단 웹을 개발하는 것을 목표로 합니다. 사용자는 웹페이지에서 카메라를 실행한 뒤, 실제 퍼스널컬러 진단처럼 다양한 색상의 가상 천을 얼굴 주변에 적용해볼 수 있습니다. 이때 버튼을 직접 누르지 않고 손동작을 통해 색상 천을 넘기거나, 자신에게 어울린다고 생각하는 색상을 선택할 수 있도록 구현할 예정입니다.

선택된 색상들은 웜톤·쿨톤, 명도, 채도, 청탁, 계절 타입 등의 기준에 따라 분석되며, 최종적으로 사용자에게 어울리는 퍼스널컬러 결과를 제공합니다.

웹에는 사용자는 퍼스널컬러가 무엇인지, 웜톤과 쿨톤의 차이, 명도와 채도의 개념 등을 이해할 수 있는 섹션도 추가할 계획입니다.

## 앞으로의 개발 계획

* 웹페이지 기본 구조 및 디자인 설계
* 카메라 기능으로 천 띄우기, 넘기기 기능 설계
* 선택된 색상 데이터들로 계절 타입 분석하는 알고리즘 구현
* +@ : 조명 상태 확인 기능, 진단 결과 신뢰도 표시, 메이크업 제품/옷 사진 색상 분석해 어울림 정도 알려주기 기능
